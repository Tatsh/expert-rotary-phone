== FUN_00034f4c mainTaskUpdate / musicSelUpdate (per-frame list scroll) ==
Reads NEGraphics touch state, drives horizontal list scroll physics.
Offsets:
  0x928  active drag touch id (-1 = none)    [also = selectedCell in ctor/update? NO:
         ctor sets 0x928=-1; update.mm treats 0x928 as "selectedCell". Here 0x928 is the
         list-drag touch-id. SAME field: -1 idle, else the captured touch id / chosen cell.]
  0xa78,0xa7c = current touch x,y ; 0xa80 = touch-moved flag (bool)
  0x92c..0x984 = scroll history ring (positions 0x930/0x940/0x950/0x960/0x970 x4 comps,
                 timestamps 0x92c.., velocity accumulators)
  0x97c  scroll velocity (float) ; 0x980 scroll offset (int)
  0x984  scroll animation phase (0 idle,1 right,2 left,3/4 settle)
  0x8f0  current column index ; 0x8f4 column count-1 bound
  0x8c0  highlight ring (bytes 0x8c0,0x8c1,0x8c2) shifted on column change; 0xff sentinel
  0xa64  screen width (scroll clamp) ; 0xa74 column stride
  Calls musicSelLoadColumnPrev/Next(this, startCol) on column change (streams jackets).
  Uses SysSePlayIntoSlot(&g_pNeSceneManager, 4) for scroll tick SE.
NOTE: 0x928 is BOTH the drag-touch-id here and, in update.mm case 2/3, the chosen cell.
  Keep as one int field m_dragTouchOrCell (ctor -1).
